# 📚 Bookora

> **Curated books for curious minds.**

Bookora is a modern, responsive online bookstore built using **HTML5**, **CSS3**, and **JavaScript**. It allows users to discover books across multiple genres, search and filter collections, save favorite books, manage a shopping cart using localStorage, and enjoy a clean, intuitive browsing experience.

This project is being built following a structured project-based development approach inspired by industry practices, focusing on clean architecture, responsive design, accessibility, performance, and maintainable code.

---

## 🌟 Features

- 📖 Browse books by category
- 🔍 Live Search
- 🎯 Category Filters
- 📚 Individual Book Details
- ❤️ Wishlist
- 🛒 Shopping Cart
- 💾 localStorage Persistence
- 👤 Login & Signup UI
- 📩 Contact Form with Validation
- 📱 Fully Responsive Design
- ✨ Modern Clean UI
